<h1>Blog Application</h1>
<blockquote>
<p>User Authentication using JWT JSON WebToken in Node.js using Express, Used Joi for user input validation, mongoose for ORM</p>
</blockquote>
<h1>Installation</h1>
<code>npm install</code>
<h1>Start an app</h2>
<code>npm start</code>
